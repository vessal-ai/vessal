# vessal-skills

Official SkillHub curated registry for the [Vessal](https://github.com/vessal-ai/vessal) Agent Runtime.

## Install a Skill

```bash
vessal skill install <name>
```

## Search for Skills

```bash
vessal skill search <keyword>
```

## Repository Structure

```
vessal-skills/
  registry.toml    — name-to-source mapping (official + third-party)
  skills/          — official skill source code
```
